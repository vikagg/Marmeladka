 •  Откройте терминал.
  •  Перейдите в корневую папку вашего проекта (my-project/).
  •  Выполните команду: npm install express cors

 Запуск сервера Node.js:
  •  В том же терминале, находясь в корневой папке проекта, выполните: node server.js
  •  Вы должны увидеть сообщение: Server listening on port 3000

 Открытие в браузере:
  •  Откройте ваш веб-браузер (Chrome, Firefox и т.д.).
  •  Перейдите по адресу: http://localhost:3000
